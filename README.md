# WumpusLite
A version of the video game Hunt the Wumpus except with 12 rooms, 1 of each hazard and 3 arrows with a range of 3 rooms.
Going into the room with the wumpus will instantly result in a game over instead of just startling the wumpus.

To move: Enter M and then a room number that is connected to the room.
To shoot and arrow: Enter S and number of rooms you want the arrow to go through. Then enter the room numbers of the rooms the arrow will fly through (if the number is invalid, the arrow will fly randomly).
